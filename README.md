pyos_rleeth is a project that provides utility functions to add or sum numbers.

## Get started

You can install this package into your preferred Python environment using pip:

```bash
$ pip install pyos_rleeth
```

To use pyos_rleeth in your code:

```python
>>> import pyos_rleeth
```

### Functions
- `add(a, b)`: Returns the sum of two numbers `a` and `b`.
  
  Example:
  ```python
  >>> pyos_rleeth.add(2, 3)
  5
  ```
- `multiply(a, b)`: Returns the product of two numbers `a` and `b`.
  
  Example:
  ```python
  >>> pyos_rleeth.multiply(2, 3)
  6
  ```
